## [1.0.0] - 2026-03-01

### 🎯 Added
- 🚀 Agent Skills 正式发布
- 📚 上海失业保险政策知识库（2025-2026）
  - 失业保险金咨询模块
  - 灵活就业社保模块
  - 4050 就业困难人员补贴模块
  - 企业稳岗补贴模块
- 💬 15 类高频问题标准问答库
- 🔗 官方资源链接列表
- 🔄 自动网页内容抓取脚本
- 🛠️ 一键安装工具 (`install.py`)
- 📦 支持多个 AI 助手集成（Cursor、Claude Code、通义灵码）

### 🔧 Technical Features
- ✅ Python 3.9+ 兼容（无额外依赖）
- ✅ 渐进式披露策略，优化 Token 使用
- ✅ 双语支持（中文 + 英文）
- ✅ 多平台支持（Windows / macOS / Linux）
- ✅ 基于官方政策文件的高准确度

### 📊 Policy Coverage
- 失业金（2305 元/月起）
- 灵活就业社保（缴费基数 7460-37302 元/月）
- 4050 补贴（约 1100-1200 元/月）
- 企业补贴（2000-1500 元/人）

## 如何报告问题 | How to Report Issues

发现 bug 或过期信息？请在 [Issues](../../issues) 中报告，包含：
- 具体问题描述
- 复现步骤（如适用）
- 预期表现 vs 实际表现
- 相关官方链接或文件

Found a bug or outdated information? Report it in [Issues](../../issues) with:
- Clear problem description
- Steps to reproduce (if applicable)
- Expected vs actual behavior  
- Relevant official links or documents
